Tworzenie i stylowanie
tabel w HTML i CSS

Znaczniki tworzące tabele w HTML

  • <table> - główny znacznik tworzący tabelę
  • <caption> - nadaje tytuł tabeli - musi być wstawiony zaraz
    po tagu <table>
  • <tr> - tworzy wiersz w tabeli
  • <th> - tworzy komórki w wierszu nagłówkowym tabeli - tekst w takiej
    komórce jest pogrubiony i wyśrodkowany
  • <td> - tworzy komórki w pozostałych wierszach

Parametryzowanie tabeli w HTML

Atrybuty

  • colspan = "3" scalenie 3 kolumn
  • rowspan = "2" scalenie 2 wierszy

<table>
<tr>
<td colspan = "3"></td>
<td rowspan = "2"></td>
</tr>
<tr>
<td> </td>
<td> </td>
<td> </td>
</tr>
</table>

    Parametyzowanie tabeli w CSS

    Jak poprawnie stworzyć obramowanie.

    • Poprawny schemat tworzenia obramowania:
    • {border: 2px solid green;}
    • px - określa grubość obramowania
    • solid - określa typ obramowania
    • green - określa kolor obramowania
    • Można także użyć subatrybutów: left, right, top, bottom:
    • {border-left: 1px solid black;}
    • {border-right: 1px dashed blue;}
    • {border-top: 1px double green;}
    • {border-bottom: 1px dotted yellow;}

    Tabele możemy stylować za pomocą różnych
    atrybutów. M.in :

    • width - szerokość
    • height - wysokość
    • background-color - kolor tła tabeli lub komórki
    • text-align - wyrównanie tekstu w poziomie
    • background-image: utl("plik.jpg") - obrazek jako tło

    Własności border collapse

    Border collapse jest używany do stworzenia połączonych obramowań

    • "rozdzielone" | "połączone"
    • border-collapse: separate | collapse {odstępy | bez odstępów}
    • W modelu połączonych obramowań sąsiadujące komórki tabel współdzielą obramowania
    • W modelu rozdzielonych obramowań sąsiadujące komórki mają swoje własne odrębne obramowania
    • Odległość między obramowaniami sąsiednich komórek:
    • border-spacing: 10px
    • Wyrównanie elementów w pionie:
    • vertical-align: top|bottom
